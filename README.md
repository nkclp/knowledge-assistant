# 📚 知识库助手

一个基于 RAG（检索增强生成）和 Agent 技术的个人知识管理工具。记录学习笔记，AI 帮你回顾和推荐下一步学习方向。

## ✨ 功能

- 📝 **写笔记**：记录你的学习内容，自动保存
- 💬 **问问题**：基于笔记内容提问，AI 智能回答
- 🎯 **智能推荐**：分析学习历史，推荐下一步学习方向

## 🛠️ 技术栈

- Python 3.14
- Streamlit（网页界面）
- DeepSeek API（大模型）
- Sentence Transformers（本地向量检索）
- NumPy（相似度计算）
